## 🚨 紧急情况应对

### 如果服务器无法启动

**症状：** 端口被占用

**解决：**
```bash
# 查找占用进程
netstat -ano | findstr :8000

# 记下 PID，然后杀死进程
taskkill /PID <PID号> /F

# 重新启动
python run_server.py
```

---

### 如果浏览器无法访问

**症状：** 页面无法加载

**解决：**
1. 检查服务器是否运行
2. 尝试 http://127.0.0.1:8000
3. 清除浏览器缓存
4. 使用无痕模式

---

### 如果文件上传失败

**症状：** 上传后显示错误

**解决：**
1. 检查文件格式（必须是 .csv 或 .json）
2. 检查文件大小（< 50MB）
3. 使用示例文件 `examples/sample_ecu_log.csv`

---

### 如果工程仪表板显示错误

**症状：** 面板显示 "Error loading data"

**解决：**
1. 这是正常的（某些组件未配置）
2. 讲解时说明："这个面板在配置 GitHub/Jenkins 后会显示数据"
3. 重点展示 Git Activity 面板（这个一定有数据）
