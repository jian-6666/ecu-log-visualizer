## 数据流向图

### 流程 1：用户上传文件

```
用户
  │
  │ 1. 选择文件
  ▼
浏览器 (index.html)
  │
  │ 2. POST /api/upload
  ▼
FastAPI (main.py)
  │
  │ 3. 调用 FileHandler
  ▼
file_handler.py
  │
  ├─ 4a. 验证文件格式 (.csv/.json)
  ├─ 4b. 检查文件大小 (<50MB)
  ├─ 4c. 生成唯一 ID (UUID)
  └─ 4d. 保存到 uploads/
  │
  │ 5. 返回 FileMetadata
  ▼
浏览器
  │
  │ 6. 显示成功消息
  │ 7. 保存 file_id
  ▼
用户看到上传成功
```

### 流程 2：查看统计信息

```
用户
  │
  │ 1. 点击"查看统计"
  ▼
浏览器 (app.js)
  │
  │ 2. GET /api/stats/{file_id}
  ▼
FastAPI (main.py)
  │
  │ 3. 调用 DataAnalyzer
  ▼
data_analyzer.py
  │
  ├─ 4a. 读取文件
  ├─ 4b. 解析 CSV/JSON
  ├─ 4c. 验证数据
  ├─ 4d. 计算统计 (min, max, mean, std)
  └─ 4e. 应用过滤器（如果有）
  │
  │ 5. 返回 StatisticsResponse
  ▼
浏览器
  │
  │ 6. 渲染统计表格
  ▼
用户看到统计结果
```

### 流程 3：查看工程仪表板

```
用户
  │
  │ 1. 打开 engineering-dashboard.html
  ▼
浏览器 (engineering-dashboard.js)
  │
  │ 2. GET /api/engineering/dashboard
  ▼
FastAPI (main.py)
  │
  ├─ 3a. 调用 GitRepository
  │   └─ 执行 git log
  │   └─ 返回提交历史
  │
  ├─ 3b. 调用 GitHubActionsMonitor
  │   └─ 请求 GitHub API
  │   └─ 返回 CI 状态
  │
  ├─ 3c. 调用 JenkinsMonitor
  │   └─ 请求 Jenkins API
  │   └─ 返回构建状态
  │
  └─ 3d. 调用 DockerMonitor
      └─ 执行 docker ps
      └─ 返回容器状态
  │
  │ 4. 聚合所有数据
  │ 5. 返回 DashboardData
  ▼
浏览器
  │
  ├─ 6a. 渲染 Git Activity 面板
  ├─ 6b. 渲染 CI/CD Status 面板
  ├─ 6c. 渲染 Docker Status 面板
  ├─ 6d. 渲染 Test Results 面板
  └─ 6e. 渲染 API Health 面板
  │
  │ 7. 30 秒后自动刷新
  ▼
用户看到实时状态
```

### 流程 4：CI/CD 自动化

```
开发者
  │
  │ 1. 修改代码
  ▼
Git
  │
  │ 2. git commit
  │ 3. git push
  ▼
GitHub
  │
  ├─ 4a. 触发 GitHub Actions
  │   │
  │   ├─ 检出代码
  │   ├─ 安装依赖
  │   ├─ 运行 flake8 (代码检查)
  │   ├─ 运行 pytest (测试)
  │   └─ 构建 Docker 镜像
  │
  └─ 4b. 触发 Jenkins (通过 webhook)
      │
      ├─ 检出代码
      ├─ 安装依赖
      ├─ 运行 flake8
      ├─ 运行 pytest
      └─ 构建 Docker 镜像
  │
  │ 5. 结果通知
  ▼
开发者收到通知
工程仪表板显示最新状态
```

## 关键概念可视化

### 什么是 API？

```
前端 (浏览器)                    后端 (服务器)
     │                               │
     │  "给我文件 123 的统计信息"      │
     ├──────────────────────────────>│
     │  GET /api/stats/123           │
     │                               │
     │                               │ 处理请求
     │                               │ 读取文件
     │                               │ 计算统计
     │                               │
     │  返回 JSON 数据                │
     │<──────────────────────────────┤
     │  {                            │
     │    "min": 10,                 │
     │    "max": 100,                │
     │    "mean": 55                 │
     │  }                            │
     │                               │
```

### 什么是 Docker 容器？

```
你的电脑
┌─────────────────────────────────────────────┐
│                                             │
│  ┌───────────────────────────────────────┐ │
│  │         Docker 容器                    │ │
│  │  ┌─────────────────────────────────┐  │ │
│  │  │  ECU Log Visualizer 应用         │  │ │
│  │  │  - Python 3.12                  │  │ │
│  │  │  - FastAPI                      │  │ │
│  │  │  - 所有依赖                      │  │ │
│  │  │  - 应用代码                      │  │ │
│  │  └─────────────────────────────────┘  │ │
│  │                                       │ │
│  │  隔离的环境，不影响主机                │ │
│  └───────────────────────────────────────┘ │
│                                             │
│  主机操作系统 (Windows/Mac/Linux)           │
└─────────────────────────────────────────────┘
```

### 什么是 CI/CD？

```
开发者写代码
     │
     ▼
提交到 Git
     │
     ▼
推送到 GitHub
     │
     ▼
自动触发 CI/CD
     │
     ├─> 运行测试 ──> ✅ 通过 ──┐
     │                         │
     ├─> 代码检查 ──> ✅ 通过 ──┤
     │                         │
     └─> 构建应用 ──> ✅ 成功 ──┤
                               │
                               ▼
                          准备部署
                               │
                               ▼
                          部署到服务器
                               │
                               ▼
                          用户可以使用
```
